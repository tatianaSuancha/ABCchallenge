## 📄 Descripción

ABC Challenge es una aplicación web educativa que presenta las letras del abecedario mediante tarjetas interactivas.

Cada tarjeta muestra inicialmente una letra y, al hacer clic sobre ella, gira para mostrar una imagen y una palabra relacionada.

El proyecto también incluye filtros para mostrar determinadas tarjetas y un contador de progreso que registra las letras descubiertas por el usuario.

Este proyecto fue desarrollado como actividad grupal para fortalecer el uso de HTML, CSS, JavaScript, GitHub y Trello.

## ✅ Características

- Visualización de las letras del abecedario mediante tarjetas.
- Cada tarjeta contiene una parte frontal y una parte posterior.
- La parte frontal muestra una letra.
- La parte posterior muestra una imagen y una palabra relacionada.
- Efecto de volteo al hacer clic sobre una tarjeta.
- Clasificación de las tarjetas mediante los atributos 
- data-tipo="vocal" y data-tipo="consonante".
- Filtro para mostrar únicamente las vocales.
- Opción para volver a mostrar todas las letras.
- Contador de letras descubiertas.
- Diseño realizado con HTML y CSS.
- Interactividad implementada con JavaScript.
- Trabajo colaborativo mediante ramas de Git y GitHub.
- Organización de tareas mediante Trello.
